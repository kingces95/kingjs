# @[kingjs][@kingjs]/[create-links][ns0]
Recursively creates npm links for all  dependent packages.

## API
```ts
createLinks()
```




## Install
With [npm](https://npmjs.org/) installed, run
```
$ npm install @kingjs/create-links
```
