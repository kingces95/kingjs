# @[kingjs](https://www.npmjs.com/package/kingjs)/poset
Exports all `@kingjs/poset.*` functionality in one package.
## Usage
```js
var poset = require('@kingjs/poset');
```
## API
- [`decode`][decode]
- [`encode`][encode]
- [`forEach`][for-each]
- [`inherit`][inherit]
## Install
With [npm](https://npmjs.org/) installed, run
```
$ npm install @kingjs/poset
```
