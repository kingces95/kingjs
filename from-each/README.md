# @[kingjs][@kingjs]/[from-each][ns0]
Generates a sequence of arrays or descriptors composed of a single element each from a set of arrays.




### Remarks


## Install
With [npm](https://npmjs.org/) installed, run
```
$ npm install @kingjs/from-each
```
